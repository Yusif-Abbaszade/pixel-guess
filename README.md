# PixelGuess — Şəkli Tap! 🎮

A modern, neon-themed pixel guessing game built with HTML, CSS, and JavaScript. Challenge yourself to guess images from pixelated hints!

## 🎨 Features

- **Multiple Game Modes**: Test your guessing skills across different difficulty levels
- **Neon UI**: Eye-catching dark theme with vibrant cyan, green, and accent colors
- **Responsive Design**: Plays smoothly on desktop and mobile devices
- **Interactive Gameplay**: Dynamic mode switching and engaging visual feedback
- **Smooth Animations**: Polished transitions and hover effects

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### How to Play

1. Open `index.html` in your web browser
2. Select a game mode using the mode switcher buttons
3. Look at the pixelated image
4. Guess what the image is
5. Earn points for correct guesses!

- **Color Palette**:

- **Glowing Effects**: Neon text shadows and glow animations
- **Responsive**: Adapts beautifully to different screen sizes

## 📝 How to Customize

Edit `index.html` to:
- Change the game title or tagline
- Adjust colors by modifying CSS variables in the `:root` selector
- Add more game images or difficulty levels
- Tweak animations and transition speeds
